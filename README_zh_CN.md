# 轻语

> 随时记录当下的想法，让碎片化的思考有处可栖。

**轻语** 是一款为 [思源笔记](https://b3log.org/siyuan/) 设计的轻量级记录插件。它借鉴了 Flomo 的即时记录理念，让你无需打开文档，就能快速捕捉灵感、心情与碎片思考。同时支持两种主题风格、多种布局模式，以及与思源日记、Flomo 的双向同步。

---

## ✨ 功能特性

### 核心记录
- **即时输入**：像发朋友圈一样快速记录想法，支持 Markdown 语法、标签、图片。
- **双布局模式**：
  - **平铺布局**：传统列表，简洁清晰，适合快速浏览。
  - **卡片布局**：三列瀑布流，直观美观，适合回顾与发现。
- **标签体系**：自动识别 `#标签` 并归类，支持层级标签管理。

### 数据同步
- **同步到思源日记**：发布内容后可自动同步到当日日记，让笔记与日记无缝衔接。
- **Flomo 同步**：绑定 Flomo 账号后，可将历史笔记全量或增量同步到思源。

### 回顾与发现
- **每日回顾**：按配置规则随机抽取过往笔记，温故知新。
- **随机漫步**：随机翻开一条历史记录，在不经意间与过去的自己相遇。
- **热力图**：可视化展示近期的记录频率，激励持续记录的习惯。

### 主题与个性化
- **原主题**：经典绿色配色，清新护眼，怀旧专属。
- **适配主题**：自动跟随思源笔记的亮/暗主题与主题色，融为一体。

## 📦 安装

### 方式一：集市安装（推荐）
1. 打开思源笔记，点击左上角 `设置` → `集市`。
2. 在搜索框输入 **"轻语"**。
3. 找到插件后点击 `下载` → `安装`。

### 方式二：手动安装
1. 下载本仓库最新 Release 的 `.zip` 文件。
2. 解压后将文件夹复制到思源笔记工作空间下的 `data/plugins/` 目录中。
3. 重启思源笔记，在 `设置` → `插件` 中启用 **轻语**。

## 🚀 使用指南

### 快速开始
1. 启用插件后，点击顶部工具栏的 **轻语图标**（或按快捷键 `⌥⌘S` / `Ctrl+Alt+S`）打开插件面板。
2. 在输入框中写下你的想法，按发送按钮或回车即可保存。
3. 使用 `#标签名` 为笔记添加标签，方便后续筛选。

### 视图切换
- 点击左侧边栏的菜单可切换：
  - **全部笔记**：查看所有记录。
  - **每日回顾**：按设定规则抽取历史笔记。
  - **随机漫步**：随机查看一条笔记。

### 设置说明
点击左侧边栏底部的 **设置** 按钮进入设置面板：

| 设置项 | 说明 |
| :--- | :--- |
| **说说视图** | 切换平铺/卡片布局，切换原主题/适配主题。 |
| **Flomo 同步** | 登录 Flomo 账号并配置同步目标（每日笔记 / 指定文档 / 说说视图）。 |
| **同步设置** | 选择日记笔记本，开启自动同步到日记。 |
| **数据管理** | 导出所有数据为 JSON，或清空本地数据。 |
| **回顾设置** | 配置每日回顾的内容范围、时间范围和每日条数。 |

---

## ⚠️ 数据安全提示

- 轻语的所有数据默认存储在思源笔记本地（`data/storage/` 下）。
- 建议定期通过设置面板的 **"导出数据"** 功能备份。
- 清空数据操作不可恢复，请谨慎操作。
