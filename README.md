
### 核心记录
- **即时输入**：像发朋友圈一样快速记录想法，支持 Markdown语法、标签、图片。
- **双布局模式**：
  - **平铺布局**：传统列表，简洁清晰，适合快速浏览。
  - **卡片布局**：三列瀑布流，直观美观，适合回顾与发现。

### 数据同步
- **同步到思源日记**：发布内容后可自动同步到当日日记，让笔记与日记无缝衔接。
- **Flomo 同步**：绑定 Flomo 账号后，可将历史笔记全量或增量同步到思源。

### 回顾与发现
- **每日回顾**：按配置规则随机抽取过往笔记，温故知新。
- **随机漫步**：随机翻开一条历史记录，在不经意间与过去的自己相遇。
- **热力图**：可视化展示近期的记录频率，激励持续记录的习惯。

### 主题与个性化
- **原主题**：经典绿色配色，清新护眼，怀旧专属。
- **适配主题**：自动跟随思源笔记的亮/暗主题与主题色，融为一体。
